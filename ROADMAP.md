# VidPeek Roadmap

VidPeek aims to be modern typed preview-generation infrastructure for media apps.

## v1.0 — Shipped

Foundation release.

- Stable CLI + Node API
- WebP preview generation
- Presets
- Safe temp dirs
- Benchmark script
- CI

## Next

Preview quality and metadata.

- Scene-change segment strategy (implemented, unreleased)
- Better JSON metadata output
- GIF and MP4 polish
- More examples
- Integration tests with FFmpeg

## Future

More media-app primitives.

- Contact sheets
- Sprite generation
- AVIF exploration
- Concurrency controls
- Richer benchmark reports

## Later

Larger possibilities.

- Worker mode
- Programmatic progress events
- Cross-platform binary packaging exploration
- Plugin-style segment strategies
